# wepy-nideshop 
[wepy](https://github.com/wepyjs/wepy)版 nideshop 微信小程序      
本项目需要配合[NideShop](https://github.com/tumobi/nideshop)商城服务端使用

## install
```bash
npm install wepy-cli -g
cd wepy-nideshop
npm install
wepy build --watch
```

## help
```bash
node_modules/.bin/eslint --fix src/pages/xxx.wpy # eslint fix file
```
